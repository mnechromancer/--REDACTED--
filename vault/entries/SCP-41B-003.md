# Item #: SCP-41B-003

**Object Class:** Euclid  ·  **Site:** Site-41B

## Special Containment Procedures

SCP-41B-003 is held on Stack I beside the intake records it was logged with and is read on the
same annual audit cycle as the reference copy [[SCP-41B-001]]. The switchboard is not to be
operated. Its incoming register is to be transcribed at each audit and the transcription filed
by the same method the reference copy uses, so that the two can be checked against each other.

## Description

SCP-41B-003 is a manual telephone switchboard, accessioned in the 1962 lot, with no external
wiring of any kind. It was brought down through the adit on the same haul as the rest of the
accession, by the road the access survey [[SCP-41B-004]] records, and has sat on Stack I since. Bench inspection confirms it terminates in nothing: the
trunk lugs are unconnected and the cabinet behind the jackfield is empty. It nonetheless
completes calls. A cord seated in any jack returns a connected line — never a voice, only the
open, patient quiet of a held circuit — to a destination the board's own register records and no
exchange can place.

The anomaly is the register, not the call. At each audit the incoming log is transcribed and
the filing method named on this record is struck out: the ⟦a1⟧. The method is not a secret; it
is the ordinary one. The reference copy [[SCP-41B-001]] states it plainly in its own containment
procedures — the same three-copy discipline every Stack I holding is filed under, named there in
full where here it is withheld.

What the audit finds is that the three copies of SCP-41B-003's register do not agree, and that
they disagree in a consistent direction: each year's third copy lists one more completed call
than the first two, addressed to a line that was not in last year's third copy and will be in
next year's first two. The board is filling its own register forward, one entry per cycle, and
the filing method is the only thing holding the three copies close enough to notice.

## Addendum 003.1 — Archivist's note

A margin note in the hand of the previous holder, R. Halloran, runs down the audit sheet:

> count the copies, not the years. the board doesn't ring out, it rings *in*, and the third
> copy is where it answers. file it by hand or it files itself.
