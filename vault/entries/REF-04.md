# REF-04 — Site-41B Personnel Directory (stale copy)

**Correspondent-site staff directory · 1966 revision · held at Site-81C for routing
inbound correspondence · NO AMENDMENTS RECEIVED SINCE 1968**

## Standing note

Receiving stations hold a directory for every site whose paperwork they process, so that
names on inbound documents can be routed, verified, and answered. This is the current
copy for Site-41B. It is eighteen years old. Amendment sheets stopped arriving after the
custody reorganization of 1968, and requests for a fresh revision return no carrier. Staff
are to use it anyway; it is the only list there is.

## Directory — Site-41B, deep-records annex (as of 1966)

**VOGEL, I. — Director.** Signs annex memoranda. No office listed on the annex blueprint
sheet attached to the 1961 revision; the omission was queried and the query was not
answered. Correspondence to the Director is to be addressed to the annex generally.

**SZE, E., Dr. — Records Officer (founding).** Author of the annex's retention practices
and of its cross-reference programme. The 1966 sheet lists Dr. Sze as *detached for
catalogue work, Stack V*. No later status is on file at this station.

**HALLORAN, R. — Records Clerk, Stack II.** Junior on the 1966 sheet. Handles intake
transcription and audit filing. Correspondence from the annex in a small, hurried hand is
usually Halloran's; route replies to the records office, not the clerk directly.

**MARSH, A. — Custodian.** On every shift roster attached to every revision of this
directory, including revisions that predate the annex's conversion. The entry is retained
as received.

**ANDRADE, T. — Liaison, Site-41 (parent).** Countersigns annex manifests on behalf of
the parent site. All pre-1968 paperwork carries the countersignature; no document received
after 1968 carries it. Correspondence to the liaison office returns undelivered.

## Routing note

Where an inbound document and this directory disagree about a name, a title, or a
posting, file the discrepancy with the document. Do not amend the directory by hand; the
directory is the station's record of what it was last told, and its staleness is itself
information.
