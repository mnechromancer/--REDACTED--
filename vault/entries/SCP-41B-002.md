# Item #: SCP-41B-002

**Object Class:** Safe  ·  **Site:** Site-41B

## Special Containment Procedures

SCP-41B-002 is the operating primer for the annex's cross-reference index and is held on
Stack I beside the intake record it serves, [[SCP-41B-001]]. It is read, not audited; the
primer is procedure, not holding.

## Description

SCP-41B-002 is the Concordance: the cross-reference index built on the catalogue mainframe to
thread each holding to every other that mentions it. A value confirmed in one file propagates,
through the Concordance, to every file that carries it — so that the archive checks itself, and
a fact stated once need not be re-entered by hand everywhere it recurs. This is the system named
only by its initial on the intake record, [[SCP-41B-001]].

The primer was kept current by the post's previous holder, whose name the cover sheet redacts:
the ⟦a1⟧. The signature stands on the intake record this primer serves, [[SCP-41B-001]].

## Addendum 002.1 — On honest threading

The Concordance, used honestly, is a convenience and nothing more: it spares the archivist the
labour of re-entering a confirmed value in every file that carries it. The primer's standing
caution — set down by the same hand that kept it — is that the convenience and the danger are
the same mechanism. A value propagated is not a value proven. The index agrees with whatever it
indexed last. Confirm against the source, then let it thread; never the other way.
