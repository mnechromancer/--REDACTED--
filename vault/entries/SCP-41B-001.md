# Item #: SCP-41B-001

**Object Class:** Safe  ·  **Site:** Site-41B

## Special Containment Procedures

SCP-41B-001 is the earliest intake record held at the deep-records annex and is to be kept
on Stack I as the reference copy against which later filings are checked. It is audited
annually and filed in triplicate. The auditor records the description verbatim and does not
reconcile it against prior years; the discrepancy, where one appears, is the datum.

## Description

SCP-41B-001 is the annex's first accession: a lot of cold records logged in 1962 and entered
into the cross-reference index then being built on the catalogue mainframe. The index — the
system that would later thread every holding to every other — is named on this record only by
its initial, the rest of the word struck out: the ⟦a1⟧.

The strike is clerical, not anomalous. The same system is set down in full on the holding this
record points to, [[SCP-41B-002]], the operating primer kept by the post's previous holder.

## Addendum 001.1 — Archivist's note

The reference copy carries a single margin note in the hand of the post's previous holder,
R. Halloran:

> start here. this is where the index begins, and where it is still honest. read 002 before
> you trust anything it threads after.
