# Item #: SCP-41B-001

**Object Class:** Safe

## Special Containment Procedures

SCP-41B-001 is to be held in a standard document vault on Stack I and audited annually. Each
audit is to be performed against the originating holding record and filed in triplicate. The
auditor is to record the description verbatim and is not to reconcile it against prior years'
filings; the discrepancy is the datum. SCP-41B-001 shares recovery provenance with
[[SCP-41B-003]] and an accession lot with [[SCP-41B-002]].

## Description

SCP-41B-001 is a recovered intake catalogued against ⟦a2⟧. The holding record describes a
single physical item; under audit, ⟦a3⟧. The direction of the shift is stable: across the
filings on hand, each year's description sits one step further along the same axis than the
last, while the item in the vault is unchanged on inspection.

At the earliest legible filing the item is recorded as ⟦a1⟧ — the same provenance the
recovery file shares with [[SCP-41B-003]]. Later filings describe the same object in terms
progressively further from that reading, though no amendment order appears in any copy. The
object class remains Safe: the anomaly is in the record, not the artifact.

## Addendum 001.1 — Audit Note

The drift is legible only across filings, never within one. An auditor reading a single
year's entry finds an ordinary description; an auditor laying three years side by side finds
a vector. The accession lot recorded above is shared with [[SCP-41B-002]], whose addenda
accrue against the same intake — a correspondence flagged but, per standing instruction,
not reconciled.
A margin note in the 1974 copy, unsigned, reads only: *count the copies, not the years.*
