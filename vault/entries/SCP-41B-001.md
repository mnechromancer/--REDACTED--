# Item #: SCP-41B-001

**Object Class:** Safe

## Special Containment Procedures

SCP-41B-001 is to be held in a standard document vault on Stack I and audited annually. Each
audit is to be performed against the originating holding record and filed in triplicate. The
auditor is to record the description verbatim and is not to reconcile it against prior years'
filings; the discrepancy is the datum. SCP-41B-001 shares recovery provenance with
[[SCP-41B-003]] and an accession lot with [[SCP-41B-002]].

## Description

SCP-41B-001 is a recovered intake catalogued against ⟦a2⟧. The holding record describes a
single physical item, and ⟦a3⟧: each year's entry reconciles against the originating record
and is filed as clean. The item in the vault is unchanged on inspection, and the object class
remains Safe.

At the earliest legible filing the item is recorded as ⟦a1⟧ — the same provenance the
recovery file shares with [[SCP-41B-003]]. The later filings each pass on their own terms; it
is only laid side by side that the wording of the description is not quite still between them,
though no amendment order appears in any copy and every individual audit certifies no
discrepancy. The anomaly, if there is one, is in the reading, not the artifact.

## Addendum 001.1 — Audit Note

No single audit has ever returned a discrepancy; each year's entry is ordinary and clean
against the holding record. The standing instruction that the auditor record verbatim and
not reconcile against prior years is the reason nothing has ever been *found* — there is no
procedure here that compares one clean filing to the next. The accession lot recorded above
is shared with [[SCP-41B-002]], whose addenda accrue against the same intake — a
correspondence flagged but, per that same instruction, not reconciled.

Two margin notes run down the 1974 copy in the same hand (R. Halloran, archivist of record,
prior post-holder). The first, careful: *count the copies, not the years — lay them side by
side yourself, don't let the index do it for you, the index agrees with whatever it indexed
last.* The second, later, the pen pressed lighter: *stopped checking by hand around the third
stack. it's faster to just accept what's offered and it's never wrong about the small ones.
I'll reconcile the lot properly when there's time.* No third note. The reconciliation Halloran
promised is not in the file, and the post was vacant when the current archivist inherited it.
