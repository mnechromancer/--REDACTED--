# REF-01 — Object Classification Primer

**Records Practice Series, Volume 1 · issued to all records stations · this copy: Site-81C reference shelf**

## Purpose

Every anomalous holding of record carries an object class. The class is not a measure of
how dangerous a thing is. It is a measure of how well its containment is understood — a
filing judgment, entered by a records officer, about whether the paperwork around a thing
can be trusted to describe it. Staff processing inbound documentation will encounter class
designations constantly and are expected to know what each one commits the record to.

## Safe

An object is classed **Safe** when its containment is understood and reliable: the thing
does what its file says it does, under conditions the file states, and has done so through
enough audit cycles that the description can be presumed current. A Safe designation is a
statement about predictability, not harmlessness. A loaded firearm in a locked drawer is
Safe. The drawer is the containment; the file describes the drawer; the description holds.

## Euclid

An object is classed **Euclid** when it is insufficiently understood for its containment
to be presumed reliable. The thing may be cooperative for years; the class records that
nobody can state *why*, and that the file describing it must therefore be treated as
provisional. Where transfer or transit documentation requires continuous escort, posture
review at every handling, and containment that "may not be presumed," the class of record
being described is this one. Most holdings that behave — but are not understood — carry it.

## Keter

An object is classed **Keter** when its containment is understood to be *inadequate*: the
procedures on file are known to fail, or to hold only at costs the Foundation cannot pay
indefinitely. Keter is not a prediction of catastrophe. It is an admission, entered in the
record, that the paperwork is losing.

## Note on usage

The class appears on intake and transfer paperwork as the "class of record." Where a
transfer slip's posture language and its class designation disagree, the posture language
governs and the discrepancy is reported. The posture language is written by the hands that
held the thing; the class is written by the hands that filed it.
